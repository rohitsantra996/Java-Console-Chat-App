# Java Console Chat Application 💬

A simple console-based chat application built using **Core Java**.  
This project demonstrates the implementation of **Client-Server Communication** using **Socket Programming**, **Multithreading**, and **I/O Streams**.

It was created as my first networking project to improve my understanding of Java backend fundamentals and real-time communication concepts.

---

# 🚀 Features

✅ Two-way real-time communication between client and server  
✅ Separate threads for reading and writing messages simultaneously  
✅ Built using Java Socket Programming (`ServerSocket` and `Socket`)  
✅ Continuous message exchange without blocking the console  
✅ Graceful connection closing using `exit` command  
✅ Beginner-friendly and lightweight console interface  

---

# 💻 Tech Stack

- **Language:** Java (Core Java)
- **Concepts Used:**
  - Socket Programming
  - Multithreading
  - BufferedReader
  - PrintWriter
  - Input/Output Streams
  - Exception Handling

# ⚙️ How to Run the Project

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/Java-Console-Chat-App.git
```

---

## 2️⃣ Open the Project Folder

```bash
cd Java-Console-Chat-App
```

---

## 3️⃣ Compile the Java Files

```bash
javac Server.java
javac Client.java
```

---

## 4️⃣ Run the Server

```bash
java Server
```

---

## 5️⃣ Run the Client (in another terminal)

```bash
java Client
```
